1. About this application

Auto-Tok is a personal automation tool that generates sorting algorithm visualization videos and publishes them to TikTok. It uses TikTok's Login Kit and Content Posting API.

2. Use of the service

This application is intended for personal, non-commercial use. By authorizing this application you allow it to post videos to your TikTok account on your behalf.

3. TikTok data

The application accesses only the minimum scopes required:

  • user.info.basic — to identify the authorized account.
  • video.upload — to upload videos to TikTok.

No TikTok user data is stored on external servers. Tokens are stored locally on the device running the application.

All videos posted are programmatically generated algorithm visualizations. No user-generated content from third parties is published.
