Privacy Policy

Effective date: June 2026

1. Information we collect

Auto-Tok requests the following TikTok scopes:

  • user.info.basic — display name and avatar, used only to confirm the correct account is authorized.
  • video.upload — used to post videos to your TikTok profile.

2. How we use your information

The access token obtained via TikTok Login Kit is stored locally on your own device. It is used exclusively to upload videos generated by this tool. No data is transmitted to any third-party server other than TikTok's official API.

3. Data storage

OAuth tokens are saved in a local file on the device running the application. This file is never uploaded or shared.

4. Data retention

You can revoke access at any time from your TikTok account settings under "Authorized apps". Deleting the local token file removes all stored credentials immediately.

5. Third-party services

This application interacts with:

  • TikTok — for OAuth authentication and video publishing.
  • Anthropic — for AI-generated algorithm descriptions (no user data is shared).
