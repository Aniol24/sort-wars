# sort-wars

Automated TikTok channel that generates and uploads sorting algorithm duel videos daily.

Two algorithms compete side by side — sound-reactive, live stats, and a winner indicator. Algorithms are generated by Claude AI, so every video is unique.

## Stack

- Python 3.11 · ffmpeg · matplotlib · numpy/scipy
- Anthropic Claude API (algorithm generation)
- Playwright (TikTok upload)
- Docker · Raspberry Pi 3 (arm64)

## Setup

```bash
cp .env.example .env
# Fill in ANTHROPIC_API_KEY and other vars

docker compose up -d
```

On first run, a browser window opens for TikTok login. The session is saved locally and reused automatically.

## Project structure

```
src/
  algorithm_factory.py   # Claude API generates sorting algorithms
  algorithm_cache.py     # SQLite cache of generated algorithms
  renderer.py            # Visual duel renderer → mp4
  audio_engine.py        # Sound-reactive audio generation
  content_generator.py   # Picks algorithms, generates captions
  tiktok_uploader.py     # Playwright-based upload
  run.py                 # Orchestrator + scheduler
config/
  config.json            # Video, audio, schedule settings
data/                    # algorithms.db + tiktok session (gitignored)
output/                  # Generated videos (gitignored)
logs/                    # Logs (gitignored)
```
